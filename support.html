Connect Canvas with an access token

  1. Open Canvas in a browser Sign in to your school’s Canvas site (the same URL you use for classes).
  2. Open Account → Settings On the left (or profile menu), go to Account, then Settings.
  3. New Access Token Under Approved Integrations, choose New Access Token. Give it a name like “GPAce” and create it.
  4. Copy the token once Canvas shows the full token only once. Paste it into GPAce before leaving the page.
  5. Import in the app In GPAce, open the Canvas tab, find your school, paste the token, and sync.

You can revoke the token anytime in Canvas → Account → Settings → Approved Integrations. GPAce stores the token on your device only.
